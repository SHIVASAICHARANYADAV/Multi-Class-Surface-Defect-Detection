# Surface Defect Detection AI

## Overview
Surface Defect Detection AI is a deep learning web application that detects defects in steel surfaces using a trained TensorFlow model.

Users can upload an image of a metal surface and the system predicts the type of defect along with confidence.

## Defect Classes
The model detects the following defects:

- Crazing
- Inclusion
- Patches
- Pitted Surface
- Rolled-in Scale
- Scratches

## Technologies Used

- Python
- Flask
- TensorFlow / Keras
- NumPy
- Pillow
- HTML / CSS
